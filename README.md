끄글 · 끝말잇기 검색 엔진(https://singrum.github.io/ggeugle)
=======================

* 22억 가지 이상의 끝말잇기 커스텀 룰 지원
* 한방단어, 유도단어, 방어단어, 루트단어 분류 알고리즘 탑재
* MCTS 알고리즘을 통한 루트전 분석 

커스텀 룰
-----
룰 설정 버튼을 클릭하면 사전, 두음법칙, 품사, 범주, 글자수, 첫글자, 끝글자 인덱스를 설정할 수 있습니다.

예시 : 구엜룰
>사전 : (구)표국대  
>두음법칙 : 표준  
>품사 : 명사  
>범주 : 일반어, 방언, 북한어, 옛말
>글자수 : 전체 선택  
>인덱스 : 앞에서1, 뒤에서1  

예시 : 쿵쿵따
>사전 : (구)표국대  
>두음법칙 : 표준  
>품사 : 명사  
>범주 : 일반어, 방언, 북한어, 옛말
>글자수 : 3  
>인덱스 : 앞에서1, 뒤에서1  

예시 : 앞말잇기
>사전 : (구)표국대  
>두음법칙 : 표준  
>품사 : 명사  
>범주 : 일반어, 방언, 북한어, 옛말
>글자수 : 전체 선택  
>인덱스 : 뒤에서1, 앞에서1  

예시 : 두샘룰
>사전 : 우리말샘  
>두음법칙 : 표준  
>품사 : 명사  
>범주 : 일반어
>글자수 : 2
>인덱스 : 앞에서1, 뒤에서1  

음절, 단어의 분류
-----------
끝말잇기에서 사용되는 단어를 **간선(Edge)** 으로 보고, 그 단어의 첫글자와 끝글자를 **정점(Node)** 으로 보면, 단어들을 다중 유향 그래프(Multidigraph)로 나타낼 수 있는데 이 그래프를 **단어 그래프** 라고 부릅니다.  
끝말잇기에서 사용하는 단어는 그 특성에 따라 공격단어, 방어단어, 루트단어로 분류할 수 있습니다.
단어 분류 이전에 **음절 분류**가 먼저 이루어집니다.  
음절 분류를 위해 **재사용룰**이라는 개념을 도입하였습니다.  
* **재사용룰** : 일반적으로 끝말잇기에서 과거에 누군가가 말한 단어를 또 말할 수 없지만, 재사용룰에서는 말한 단어를 또 말할 수 있습니다.
 
* **승리음절** : 재사용룰에서 승리음절이 주어진 경우 승리전략이 존재합니다.
* **패배음절** : 재사용룰에서 패배음절이 주어진 경우 상대방에게 승리전략이 존재합니다.
* **순환음절** : 재사용룰에서 순환음절이 주어진 경우 누구에게도 승리전략이 존재하지 않습니다.

기본 끝말잇기 룰(단어를 재사용할 수 없는 룰)에서 순환음절이지만 승패 여부가 명확한 음절의 예시는 다음과 같습니다.  
  
맴 : 맴맴 - gg  
증 : 증휼 - 휼궤 - 궤휼 - gg   
욕 : 욕중관수욕 - 욕식 - 식욕 - gg  

이러한 음절을 분류하기 위해 특정 단어 그래프 내에서 특정 패턴을 필터링 하여 승리음절과 패배음절에 추가합니다. 필터링하고 남은 음절을 **루트음절** 이라고 합니다.

마지막으로 루트 음절로 구성된 유도 부분 그래프(Induced Subgraph)를 강한 연결 요소(Strongly Connected Component)로 분해했을 때 나오는 가장 큰 요소를 **주요루트음절** 라고 하고, 나머지 요소를 **희귀루트음절** 이라고 합니다.

검색
----
음절을 클릭하거나 검색창에 음절을 입력하면, 그 음절로 시작하는 단어들을 공격단어, 루트단어, 방어단어로 분류하여 보여줍니다.

* **공격단어** : 패배음절로 끝나는 단어
* **방어단어** : 승리음절로 끝나는 단어
* **루트단어** : 루트음절로 끝나는 단어


분석
---------
끝말잇기에서 서로 지지 않기 위해서 루트단어를 말하게 되며, 이것을 **루트전** 이라고 하며 현시대 끝말잇기의 메인 컨텐츠입니다. 루트전에서 이기기 위해선 복잡한 수를 읽어야 합니다. 루트전의 승리전략을 찾는 문제는 [**PSPACE-Complete**](http://www.google.co.kr) 이며 인공지능 기법을 통해 컴퓨터로 해결할 수 있습니다.  
끄글은 [몬테카를로 트리 탐색](https://en.wikipedia.org/wiki/Monte_Carlo_tree_search)으로 루트전을 분석하는 도구를 제공합니다.


통계
---------
통계 버튼을 클릭하면 설정된 룰에 대한 대푯값을 볼 수 있습니다. 대푯값은 **단어 수**, **음절 수**, **승리음절 수**, **패배음절 수**, **주요루트음절 수**, **루트 복잡도**(루트단어 수 / 루트음절 수) 가 있으며 룰을 나타내는 지표가 됩니다.
